Windows 7 64bitなどの64ビットOSでは動作しますか？
64ビット正式対応は，バージョン0.8.3からです． 64ビットOS用バイナリの配布もこのバージョンから開始されました．
Windows XP以降では動作しますか？
正式対応は，バージョン0.6.8からです． それより前のバージョンでは，管理者権限が ないと動作しないとか，設定ファイルや一時ファイルが変なフォルダに作成されるなどの 問題点があります．
なお，バージョン0.4までは，再生すると固まるという報告があります． また，バージョン0.6.0には，バグがあるため，Windows XPで起動しないことが分かっています．
一方で，バージョン0.6.7以降からWindows XPより前のOSは非サポートとなっています．
Mac OS Xには対応していますか？
Mac OS Xネイティブ版がリリースされています． これまではCarbon版のみでしたが，バージョン0.8.3からCocoa版もリリースされています． まだ大規模な動作チェックは完了していませんが，Mac OS X 10.6以降の環境では，Cocoa版がより安定していると思います． なお，現在のCarbon版は10.4以降のみに対応していますので，10.3以前では動作しません．
Linuxで起動しません．
バイナリバージョンをインストールした場合， libcのバージョンやgtkのバージョンの不一致により起動しないことがあります． こういった場合は再コンパイル & 再インストールを行って下さい． SRPMやdebファイルを使用するのが簡単だと思います．
また，spwave本体とプラグインのlibcのバージョンが異なっている場合でもファイルの読み込みに失敗します． 「どんなファイルも読み込めません」も参考にして下さい．
