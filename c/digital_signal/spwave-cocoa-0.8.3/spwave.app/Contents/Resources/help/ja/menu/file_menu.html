ファイルメニュー

ユーザーズガイド > メニューリファレンス > ファイルメニュー

ファイルの読み込み・保存，アプリケーションの終了などを行います．

開く...

現在のウィンドウにファイルを読み込みます． クリックすると，ファイルダイアログが開きますので，所望のファイルを選択して下さい． このとき，「ファイルの種類」のメニューにより，どのように開くかを選ぶことができます． 以下の項目が選択可能です．

  • Auto Detect - 自動的にフォーマットを選択してファイルを開きます． 判定できない場合はフォーマットを選択するダイアログが開きます．
  • Default - 自動的にフォーマットが判定されない場合に， デフォルトのフォーマットでファイルを開きます．
  • Frequency Data - 自動的にフォーマットが判定されない場合には，周波数領域のデータとして扱います． フォーマットを選択するダイアログにおいて選択した周波数の半分の長さのデータとして扱われます．

新規に開く...

開く...と同じですが，新しいウィンドウにファイルを読み込みます．

波形生成...

正弦波，方形波，鋸歯状波などの音声ファイルを新規に生成します． 録音もこのメニューから実行することができます． 詳しくは，波形生成ダイアログを参照して下さい．

保存

ファイルに上書き保存します． ファイルの名前がまだ決まっていない場合には，名前を付けて保存...と同じ動作となります． 現在のファイルのフォーマットによっては，上書き保存できないことがありますので， できなかった場合は，ファイルのフォーマットを変換するか，別のフォーマットで保存して下さい， なお，保存できた場合でも，フォーマットによっては完全に元のファイルと同じになるわけではないので注意して下さい． MP3など非可逆圧縮が施されている場合には，編集・上書き保存を繰り返すと少しづつ劣化していきます．

名前を付けて保存...

現在のファイルを別の名前で保存します． クリックすると，ファイルダイアログが開きますので，ファイル名を入力して下さい． このとき，「ファイルの種類」のメニューにより，どのフォーマットで保存するかを選択できます． 表示されるフォーマットの数は，インストールされている出力プラグインの数に依存します．

「By Extension」を選択すると，拡張子からフォーマットを判定します． 例えば，WAVEファイルで保存したい場合は".wav"という拡張子を， AIFFファイルで保存したい場合は".aif"という拡張子を付けます． うまく判定できなかった場合は，別の拡張子を使用するか， 「ファイルの種類」メニューからフォーマットを選択して下さい． ただし，「ファイルの種類」メニューの中に表示されるものが 対応しているフォーマットの全てですので，この中にないフォーマットでは保存できません． 例えば，MP3ファイルで保存しようとして".mp3"を付けた場合でも， 「MPEG 1.0 Layer III」などの表示がメニューの中にない場合はMP3では保存できないことになります．

ラベル

  • ラベルを開く...
    ラベルを現在のウィンドウに読み込みます．
  • ラベルを保存
    現在のウィンドウのラベルを上書き保存します． ファイルの名前がまだ決まっていない場合には，ラベルを名前を付けて保存...と同じ動作となります．
  • ラベルを名前を付けて保存...
    現在のウィンドウのラベルを別のファイル名で保存します．
  • ラベルをクリア
    現在のウィンドウのラベルを消去します．
  • 範囲ラベルを開く...
    範囲ラベルを現在のウィンドウに読み込みます．
  • 範囲ラベルを保存
    現在のウィンドウの範囲ラベルを上書き保存します． ファイルの名前がまだ決まっていない場合には，範囲ラベルを名前を付けて保存...と同じ動作となります．
  • 範囲ラベルを名前を付けて保存...
    現在のウィンドウの範囲ラベルを別のファイル名で保存します．
  • 範囲ラベルをクリア
    現在のウィンドウの範囲ラベルを消去します．

印刷先

印刷先のプリンタもしくは印刷先のファイル形式を選択します．

ページ設定...

ページ設定ダイアログを開きます．

印刷...

印刷ダイアログを開きます．

閉じる

現在のウィンドウを閉じます． 保存されていないデータなどは全て破棄されますので注意して下さい．

終了

spwaveを終了します． 保存されていないデータなどは全て破棄されますので注意して下さい．
