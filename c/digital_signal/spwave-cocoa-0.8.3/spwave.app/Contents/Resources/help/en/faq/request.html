I found bugs!
Please send mail to me. Please include the following information.
  • Version of spwave (or the file name of the downloaded archive)
  • Version of the operating system (e.g. Windows XP SP2, Mac OS X 10.6.8, etc.)
  • Symptoms in delail
  • Version of the plugin if the bug is associated with a particular file. e.g. the bug is related to writing MP3 files.
Including `spwave' in the subject of the e-mail is also helpful for me.
