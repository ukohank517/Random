View Menu

User's Guide > Menu Reference > View Menu

Zooming in/out the displayed area, changing the time axis format, etc.

Zoom In

Zoom in the displayed area. The zooming in is centered at the cursor position.

Zoom Out

Zoom out the displayed area. The zooming out is centered at the cursor position.

Zoom Full Out

Zoom full out.

Zoom Region

Zoom in the selected region.

Time Format

Change time axis format. This setting will be valid for all windows.

  • Second
  • Milli Second
  • Point
  • Separated Second The format is ``Hour:Miniute:Second''.

Zoom In Vertical Axis

Zoom in vertical axis.

Zoom Out Vertical Axis

Zoom out vertical axis.

Zoom Full Out Vertical Axis

Zoom full out vertical axis. This is the initial zooming state.

Align Vertical Axis

Align the vertical axis of other windows with the current vertical axis.

Display Information Area

Display information area to show label information.

Draw Detail

Draw detail of waveform. This setting makes the drawing speed slower. Delailed drawing is sometimes invalied when the file is very long.

Draw Labels

Draw labels. The waveform will be hidden by labels when many labels exist. In this case, turning off drawing labels is useful.

Spectrogram

  • No Spectrogram
    Draw normal waveform.
  • Wide Band Spectrogram
    Draw wide band spectrogram.
  • Narrow Band Spectrogram
    Draw narrow band spectrogram.
  • Smoothed Spectrogram
    Draw narrow band smoothed spectrogram by cepstrum. Smoothed rate depends on Lifter Length in Analysis Dialog.
  • Update Spectrogram
    Redraw spectrogram.
  • Spectrogram Setting...
    Change the spectrogram setting by Spectrogram Dialog.
