Spectrogram Dialog

User's Guide > Dialog Reference > Spectrogram Dialog

Change the setting of a spectrogram. Changing a value is immediately reflected to the spectrogram drawing without pressing OK button.

Range (dB)
Range of the spectrogram. This range subtracted from Limiting Threshold gives the minimum spectrum value. Spectrum values below the minimum value will be drawn with the same color as in the minimum value.
Limiting Threshold (dB)
Limiting Threshold in dB. When the spectrum amplitude exceeds this value, same color as in the threshold will be used in drawing.
Draw Simplified Spectrogram
Although turning off this value makes drawing speed slower, drawing will be slightly accurate.
Draw Spectrogram with Gray Scale
Draw spectrogram with gray scale. I recommend this setting to conservative speech researchers who love gray scale spectrogram :-)
