Page Menu

User's Guide > Menu Reference > Page Menu

Moving displayed area, selecting a region, etc.

Forward

Forward the displayed area in 1/4 of the area.

Backward

Backward the displayed area in 1/4 of the area.

Head

Go to the head of the file.

Tail

Go to the tail of the file.

Next Window

Search the next window from the current window list and set the window into the foreground.

Previous Window

Search the previous window from the current window list and set the window into the foreground.

Align

Align the displayed area of other windows with the displayed area of the current window.

Select

Select the current displayed area.

Select All

Select the region from head to tail.

Select Same Region

Set the regions of the other windows same as the region of the current window.

Select Next Channel

Select the next channel with same length as the current selection. When all channels are selected, the first channel will be selected. When the last channel is selected, all channels will be selected.
