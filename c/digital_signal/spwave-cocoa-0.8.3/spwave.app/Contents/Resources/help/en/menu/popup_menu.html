Popup Menu

User's Guide > Menu Reference > Popup Menu

Popup menu will be displayed by clicking the right button or Control+click on Mac OS in the waveform area. The popup menu includes menus frequently used and menus often used in the waveform area.

Play Selected Region

Play the selected region. Same as clicking the middle mouse button.

Play Window Contents

Play the current displayed area.

Play Entire File

Play the entire file.

Stop Processing

Stop the current processing.

Zoom In

Same as Zoom In.

Zoom Out

Same as Zoom Out.

Zoom Full Out

Same as Zoom Full Out.

Zoom Region

Same as Zoom Region.

Align

Same as Align.

Select Same Region

Same as Select Same Region.

Select From Here

Select from here to the tail of the file.

Select To Here

Select from the head of the file to here.

Edit

  • Cut
    Cut the selected region and copy the data into the clipboard.
  • Copy
    Copy data of the selected region into the clipboard.
  • Paste
    Paste the contents of the clipboard. The target data will be replaced with the clipbard contents.
  • Mix
    Mix the contents of the clipboard. The target data will be mixed with the clipbard contents.
  • Insert
    Insert the contents of the clipboard.
  • Replace
    Replace the target data with the contents of the clipboard.
  • Change Value...
    Change value at the cursor location.
  • Insert Pause...
    Insert pause to the cursor location.

Label

  • Insert Simple Label
    Insert a simple label which includes only time information.
  • Insert Label with Value
    Insert a label which includes an amplitude value at the current cursor location. The amplitude is taken from the currently selected channel or the first channel in no channel selection.
  • Insert Label...
    Insert a label. Label Dialog will popup, and press OK after fill in desired items.
  • Change Label...
    Change label information closest to the cursor. Label Dialog will popup, and press OK after fill in desired items.
  • Erase Label
    Erase a label closest to the cursor.
  • Erase Label in Selected Region
    Erase labels in the selected region.
  • Select Between Labels
    Select an interval between two labels when the cursor is located between two labels.
  • Set Region Label
    Set the current selected region as a region label.
  • Set Region Label As Current Region
    Set the region label closest to the cursor as a current selection.
  • Concatenate Region Labels
    Concatenate the region label closest to the cursor with the adjacent region label.
  • Divide Region Label
    Divide the region label into two region labels at the cursor location.

Analysis

  • Analysis of Selected Region
    Analyze the selected region with parameters of Analysis Dialog. Too long region cannot be analyzed.
  • Wide Band Analysis
    Wide band analysis of the selected region is executed with parameters of Analysis Dialog. The selected region is adjusted so as to fit to the internal analysis frame whose default value is 8 ms. Wide band analysis gives high time resolution and low freuqnecy resolution.
  • Narrow Band Analysis
    Narrow band analysis of the selected region is executed with parameters of Analysis Dialog. The selected region is adjusted so as to fit to the internal analysis frame whose default value is 51.1 ms. Narrow band analysis gives low time resolution and high freuqnecy resolution.
