Format Dialog

User's Guide > Dialog Reference > Format Dialog

This dialog will be displayed when spwave cannot detect a format of the file automatically. spwave considers the file as a raw file or a text file. Since raw files cannot include information of the data such as sampling frequency , number of channels, and bits/sample, you have to select this type of information.

Some file formats such as WAV, AIFF and MP3 can select a type of a codec. If spwave doesn't recognize the codec, this dialog is also displayed.

File Format
Type of file format. For more information, see File Formats.
Bits/Sample
Bits per sample of the input file. When the selected file format is u-law or a-law, this value is ignored. spwave converts bits/sample into 16 bits/sample internally when the file has a bits/sample lower than 16 bits.
Select 64, if the file includes floating point numbers.
Sampling Frequency
Sampling frequency of the input file.
Number of Channels
Number of channels of the input file.
