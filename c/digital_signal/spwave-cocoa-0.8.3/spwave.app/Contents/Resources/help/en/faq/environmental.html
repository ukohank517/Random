Does spwave support 64 bit OS such as Windows 7 64 bit?
Official support for 64 bit OS was added in version 0.8.3.
Does spwave support Windows XP+?
Official support for Windows XP+ was added in version 0.6.8.
On the other hand, support for OS before Windows XP such as Windows 98/Me/2000 was abondoned in version 0.6.7.
