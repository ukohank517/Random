File Menu

User's Guide > Menu Reference > File Menu

Opening files, saving files, quiting spwave, etc.

Open...

Load a file into the current window. Select the desired file from the file dialog. You can change the attribute of opening in File Type. The following items are available in File Type.

  • Auto Detect - detects a file format automatically. The format dialog will be displayed when spwave cannot detect the format.
  • Default - spwave loads the file using the default format when spwave cannot detect the format.
  • Frequency Data - loads the file as frequency data when spwave cannot detect the format. The end point of the file is considered as half size of sampling frequency in format dialog.

Open New...

Same as Open... except to create new window when opening a file.

Generate...

Generate a new file such as a sine wave, square wave, sawtooth wave, and so on. You can also perform recording by this menu. For more information, see Generate Dialog.

Save

Overwrite the current file to the original file. When the file name is not decided yet, it is same as Same As....

Save As...

Save the current file with another file name. The file format can be chosen by `File Types'. The number of formats displayed in `File Types' depends on the number of plug-ins installed.

When you select `By Extension' in `File Types', the file format is settled by the extension of the file name. For example, the extension `.wav' means WAVE file format, `.aif' means AIFF file format. When spave cannot detect the file format from the file name, use another extension or select the desired file format in `File Types'. You cannot save as the file format that is not displayed in `File Types'. For example, if items such as ``MPEG 1.0 Layer III'' are not in the list, you cannnot save as MP3 format.

Label

  • Open Label...
    Load a label file into the current window.
  • Save Label
    Overwrite labels of the current window to the original label file. When the file name is not decided yet, it is same as Save Label As....
  • Save Label As...
    Save labels of the current window with another file name.
  • Clear Label
    Clear labels of the current window.
  • Open Region Label...
    Load a region label file into the current window.
  • Save Region Label
    Overwrite region labels of the current window to the original region label file. When the file name is not decided yet, it is same as Save Region Label As....
  • Save Region Label As...
    Save region labels of the current window with another file name.
  • Clear Region Label
    Clear region labels of the current window.

Print Target

Select target printers or target file formats.

Page Setup...

Open the page setup dialog.

Print...

Open the print dialog.

Close

Close current window.

Exit

Quit spwave.
