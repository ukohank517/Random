+ Daily Report
    + Describe: アマログより商品在庫一覧をダウンロードし、これらのファイルのまとめをしてくれるアプリです。(知り合いに頼まれて作ったしょぼいプログラム。)当日の在庫数と前日の在庫数を比較し、当日の売り上げ状況を確認します。なお、同じASINコードに対しても違うSellerであれば違う商品として判別します。

        + today: アマログよりダウンロードした当日の売り上げ情報を入れるディレクトリ。

        + test.py: 上記ディレクトリ、ファイルが存在する前提で当プログラムを実行してください。
        + consult.xlsx: プログラムを実行した後生成されます。Daily.xlsxのtodayを追加したバージョンです。（テスト段階では上書きしません）
